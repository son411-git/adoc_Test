== Isolated Danger Beacon

=== 정의
An isolated danger beacon is a beacon erected on an isolated danger of limited extent, which has navigable water all around it.

*코드*: `IsolatedDangerBeacon`  
*별칭 (Alias)*: `BCNISD`  
*정의 출처*: IHOREG (ID: 706)  
*Feature Use Type*: `geographic`  
*지오메트리 타입*: `point`
