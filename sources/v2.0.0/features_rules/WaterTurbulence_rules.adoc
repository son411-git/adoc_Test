===== Remark
- 쇄파(breakers)를 인코딩하는 경우 쇄파를 야기하는 것(예: Underwater/Awash Rock)과 동일한 위치에 인코딩
- #F "Water Turbulence" 피처는 #F "Depth Area" 또는 #F "Unsurveyed Area" 피처와 중첩
- Point와 Line, Surface Type 모두 인코딩 가능하지만 Point Type이 원칙
