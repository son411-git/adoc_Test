===== Remark
- #F "Range system"의 명칭은 각각의 그룹화로 표현되는 용도에 따라 입력 (예시 참조)
- 관련된 피처들은 no-geometry 피처인 #F "Range System"와 #R "Range System Aggregation" 관계 설정 +
   ※ Relation 페이지 참조
- 향후 #R "Range System Aggregation" 구성 시 관련 항로표지까지 포함하도록 기준 협의 필요
