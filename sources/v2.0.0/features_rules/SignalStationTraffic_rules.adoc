===== Remark
- “Signal station traffic”는 건물이나 구조물과는 상관없이 신호소 자체의 기능을 설명하기 위해서만 사용, 해당 서비스를 제공하는 건물이나 구조물을 인코딩해야 한다면 적절한 피처(Building, Landmark 등)를 사용하여 인코딩하고 #R "Structure/Equipment" 관계 설정 +
   ※ Relation 페이지 참조
